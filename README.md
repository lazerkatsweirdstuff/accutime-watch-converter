# accutime-watch-converter
Convert a video to the format acceptable for accutime educational watches.

Has been tested on [the minecraft one](https://www.amazon.com/Accutime-Minecraft-Educational-Learning-Touchscreen/dp/B0BMK86R2J?ref_=ast_sto_dp) which i got at target lol

# How to use

## You will need: 

Screwdrivers

MicroSD (Needs to be blank)

## Instructions:

1: Turn it off by turning it over and flicking the switch to off 

2: Unscrew the 4 screws on the back

3: Wedge it open from any side

4: Very carefully take the white framey looking bit out

5: Carefully pull the battery off

6: Locate the MicroSD slot

7: Insert your MicroSD into it.

8: Put the battery back
