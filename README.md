# accutime-watch-converter
Convert a video to the format acceptable for accutime educational watches.

Has been tested on [the minecraft one](https://www.amazon.com/Accutime-Minecraft-Educational-Learning-Touchscreen/dp/B0BMK86R2J?ref_=ast_sto_dp) which i got at target lol

# How to use

## You will need: 

Accutime Watch (Fully Charged)

Screwdrivers

MicroSD (Needs to be blank)

MicroSD Reader

## Instructions:

1: Turn it off by turning it over and flicking the switch to off 

2: Unscrew the 4 screws on the back

3: Wedge it open from any side

4: Very carefully take the white framey looking bit out

5: Carefully pull the battery off

6: Locate the MicroSD slot

7: Insert your MicroSD into it.

8: Put the battery back 

9: Put the white framey bit back and move the battery around so it fits inside the frame

10: Put the back cover on

11: Flick the switch to on

12: Hold the button on the side to power it on

13: Wait for the boot sequence to finish

14: Open the settings app, then click the right button until you get to the "Storage" option

15: Press "Set"

16: Press "card"

17: If you have properly inserted the MicroSD, this will say "Set to Memory Card successfully" and if you haven't, "Please insert memory card". If you get this, repeat steps 1-16 but try and push the card in more.

18: After you've successfully done all of these steps, your MicroSD will now be ready for video loading. 

19: Repeat steps 1-5

20: Take the MicroSD out of the watch, and insert into your MicroSD Reader. Insert into computer if not already. You will see a file system like this:
